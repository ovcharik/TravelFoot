# TravelFoot

Сервис формирования маршрута для пешей прогулки в городе.


## Установка

Зависимости:

  - `nodejs`, version >= 0.10.25
  - `npm`, version >= 1.3.24
  - `mongodb`, version >= 2.4.9

Первый запуск приложения:

    $ git clone git@github.com:ovcharik/TravelFoot.git
    $ cd TravelFoot
    $ npm install
    $ npm start
